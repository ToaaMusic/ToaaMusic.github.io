# Linux 相关合集前言

这个合集记录了我从零开始使用 Archlinux （以下简称为Arch）之后学习到的 Linux 相关知识点。
由于我日常主要使用 Arch，因此有些细节可能会只适用于Arch，但我尽量让文章适配通用的 Unix 操作系统。

阅读本合集文章之前，请注意以下事项：

- 我的大多数命令示例都会习惯性地以 Arch 为准，如果你使用的是其他发行版，还请自行寻找办法转换到你所使用的发行版所对应的命令，比如我写：

```bash
sudo pacman -S nvim
```

那么你应该知道在 Debian 下对应：

```bash
sudo apt install nvim
```

之后类似的情形我不会赘述

- 由于相关生态的可变性，尤其是 Arch 这种滚动更新，我不敢保证一些信息不是过时的，因此请留意我每篇文章下面的参考文献，任何时候都以它们为准

但不代表我写的文章是完全不可信的，一般地，我会把过时信息标注为“out of date”或直接更改，除非我不再使用 Linux 作为日常操作系统，届时我也会直接将这个合集标注为废弃，但未来几年是不可能的
我的文章也不是完全没有意义的，我会尽量让文章不是一味地翻译英文文档，而是将我在实践过程中学到的东西，用自己的逻辑表述出来

因此请不要把我的文章当作权威教程，它们只提供心智模型

- 请不要犯懒或过于依赖AI，而是多动动手指，点入官方文档链接

首先，AI的回答存在幻觉，你可以利用它过滤到准确的官方文档链接，或者让它 Web Search 之后给你一个总结，否则不要盲目相信AI的回答

其次，尽管AI如此方便，最上游的文档始终是最靠谱，最全面的信息来源，如果你觉得英文文档阅读困难，请使用类似“沉浸式翻译”的浏览器插件以辅助阅读

同样，本合集的任何文章，任何一个段落，都不会是AI生成或润色过的，全部由我亲手书写。
